# SO-101 조립 가이드

<Callout type="info">
이 문서는 HuggingFace LeRobot SO-101 공식 가이드를 기반으로, RoboSEasy 스타일로 재구성한 한국어 조립 안내서입니다.
</Callout>

## 1. 개요

SO-101은 6축 오픈소스 로봇 팔로, 3D 프린팅 부품과 STS3215 서보 모터를 사용하여 직접 조립할 수 있습니다.

## 2. 준비 단계

### 2.1 부품 소싱 및 3D 프린팅

- [SO-ARM100 README](https://github.com/TheRobotStudio/SO-ARM100)에서 부품 목록(BOM)과 3D 프린팅 파일을 확인하세요.
- 3D 프린팅이 처음이거나 프린터가 없다면, 주변에 출력 대행을 문의하세요.

<Callout type="warning">
3D 프린팅 부품의 서포트(지지대)는 조립 전 반드시 제거해야 합니다. 작은 드라이버를 사용하면 쉽게 제거할 수 있습니다.
</Callout>

### 2.2 LeRobot 설치

- [설치 가이드](../software/installation)를 참고하여 LeRobot을 설치하세요.
- Feetech SDK도 추가로 설치해야 합니다.

```bash
pip install -e ".\[feetech\]"
```

## 4. 모터 설정

### 4.1 USB 포트 찾기

MotorBus를 USB로 연결한 후, 아래 명령어로 포트를 확인하세요.

```bash
lerobot-find-port
```

출력 예시:
```
Finding all available ports for the MotorBus.
['/dev/tty.usbmodem575E0032081', '/dev/tty.usbmodem575E0031751']
```
각 포트가 어떤 팔(Leader/Follower)에 해당하는지 구분하세요.

### 4.2 모터 ID 및 Baudrate 설정

각 모터는 고유 ID가 필요하며, 통신을 위해 동일한 Baudrate로 설정해야 합니다.

#### Follower 예시

```bash
lerobot-setup-motors \
    --robot.type=so101_follower \
    --robot.port=/dev/tty.usbmodem585A0076841
```

#### Leader 예시

```bash
lerobot-setup-motors \
    --teleop.type=so101_leader \
    --teleop.port=/dev/tty.usbmodem575E0031751
```

<Callout type="tip">
각 모터를 하나씩 연결하여 ID와 Baudrate를 설정하세요. 케이블 연결 상태를 반드시 확인하세요.
</Callout>

## 5. 캘리브레이션

리더와 팔로워가 동일한 위치값을 갖도록 캘리브레이션이 필요합니다.

#### Follower 캘리브레이션

```bash
lerobot-calibrate \
    --robot.type=so101_follower \
    --robot.port=/dev/tty.usbmodem58760431551 \
    --robot.id=내_팔로워_이름
```

#### Leader 캘리브레이션

```bash
lerobot-calibrate \
    --teleop.type=so101_leader \
    --teleop.port=/dev/tty.usbmodem58760431551 \
    --teleop.id=내_리더_이름
```

<Callout type="info">
캘리브레이션 시 모든 관절을 중앙 위치로 맞춘 후, 각 관절을 끝까지 움직여야 합니다.
</Callout>
